## Установка проекта
Клонирование проекта SSH:
```sh
$ git clone git@github.com:prstVanya/portfolio.git
```

Установка зависимостей:
```sh
$ npm i
```
___

## Запуск проекта
Запуск проекта:
```sh
npm run start
```

development сборка:
```sh
npm run build:dev
```

production сборка: 
```sh
npm run build
```
____

## Проверка линтинга
Для проверки TypeScript-файлов используется ESLint с конфигурацией airbnb-base.

Проверка ts-файлов:
```sh
npm run lint:ts
```

Исправить ts-файлы:
```sh
npm run lint:ts:fix
```

Для проверки CSS-файлов используется Stylelint.

Проверить css-файлы:
```sh
npm run lint:css
```

Исправить css-файлы:
```sh
npm run lint:css:fix
```
